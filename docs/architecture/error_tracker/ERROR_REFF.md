# Common Error Logs

> To Quick Debugging Purpose different Section of the code is refferenced with Error Code ID, these code is assigned for unexpected or unhandled errors.

| ErrorID         | Section    | Last Reg | Description                             |
| --------------- | ---------- | -------- | --------------------------------------- |
| `ER1601-ER1650` | `services` | ER1608   | collection services error               |
| `ER1901-ER1950` | `storage`  | ER1901   | File Folder Creation and Deletion Error |
